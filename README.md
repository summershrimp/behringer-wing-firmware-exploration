# behringer-wing-firmware-exploration
## Accessing Firmware Download Mode
Pressing the tiny button between the USB-B and first network port and then booting starts the CPU in download mode.

USB-Device Description:
```
SE Blank RIGEL:

  Location ID:	0x02110000
  Connection Type:	Removable
  Manufacturer:	Freescale SemiConductor Inc
  Serial Number:	Not Provided
  Link Speed:	480 Mb/s
  USB Vendor ID:	0x15a2
  USB Product ID:	0x0061
  USB Product Version:	0x0001
```
